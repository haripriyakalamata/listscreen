# listscreen
react native login screen

Usage
------------------
<h3>Example</h3>

<pre>
import LoginScreen from '@haripriya.kalamata/loginscreen';  
</pre>

    
        
       <LoginScreen    
             backend_fetch_link='your-url'
          />
         
 
 Check out the <a href="https://snack.expo.io/rkPagvsWB">example project</a> for more examples.
